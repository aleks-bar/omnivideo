== Description ==

Вывод видео с различных платформ в одном блоке. Поддерживает YouTube, ВК.Видео, RuTube, Дзен и другие. Автоматическое получение постеров, гибкая настройка стилей, а также микроразметка для улучшения SEO.

== Installation ==

1. Upload `wp-omnivideo` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
